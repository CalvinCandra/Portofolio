# React + Vite

Templat ini menyediakan pengaturan minimal untuk membuat React bekerja di Vite dengan HMR dan beberapa aturan ESLint.

Saat ini, ada dua plugin resmi yang tersedia:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md): Menggunakan [Babel](https://babeljs.io/) untuk Fast Refresh.
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc): Menggunakan [SWC](https://swc.rs/) untuk Fast Refresh.

## Cara Menjalankan Proyek

1. Clone project ini dengan perintah berikut:
   ```bash
   git clone https://github.com/CalvinCandra/Portofolio.git
   ```
2. Jalankan perintah di bawah untuk menginstal NPM yang dimana akan menginstall dependensi diperlukan:
   ```bash
   npm install
   ```
3. Jalankan perintah di bawah untuk menjalankan project :
   ```bash
   npm run dev
   ```
   Kemudian buka http://localhost:5173 di browser untuk melihat hasilnya.
